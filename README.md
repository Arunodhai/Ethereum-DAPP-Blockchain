# Ethereum-based Decentralized Application (DApp) Project

## Overview
This decentralized application (DApp) interacts with the Sepolia Ethereum blockchain, allowing users to send and retrieve messages. Built with React for the frontend and Express/Node.js for the backend, Infura serves as the provider for seamless connectivity. Metamask integration enhances user experience by facilitating secure account connections.

## Key Features
- **Message Interaction:** Users can send messages to the blockchain and retrieve them.
- **Technology Stack:** React for the frontend, Express/Node.js for the backend, and Infura for blockchain integration.
- **Metamask Integration:** Ensures secure and user-friendly account connectivity.
